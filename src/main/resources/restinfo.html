HoneyBee Portal For Naturopathy

WebService

For the time being, only lists of some elements are available. Specific webservices to manipulate data would be implemented on demand!

Resources are produced as JSON objects

Example: https://honeybee-otika.rhcloud.com/rest/resources/plants/ordered where /rest/resources/plants/ordered is the URL suffix

List of Prescriptions

You can get a list of prescriptions through this URL suffix: /rest/resources/prescriptions/ordered

List of Plants

You can get a list of ingredients through this URL suffix: /rest/resources/plants/ordered

List of Ingredients

You can get a list of ingredients through this URL suffix: /rest/resources/ingredients/ordered

List of Stores

You can get a list of stores through this URL suffix: /rest/resources/stores/ordered

List of Honeys

You can get a list of honeys through this URL suffix: /rest/resources/honeys/ordered
